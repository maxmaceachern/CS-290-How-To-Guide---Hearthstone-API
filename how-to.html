Getting Started with the Hearthstone: Heroes of Warcraft API

This guide aims to highlight the core functionality of the Hearthstone API, developed by omgvamp. Hearthstone: Heroes of Warcraft is a digital collectable card game developed by Blizzard Entertainment. This particular API, in spite of its very generic and official sounding name, is NOT developed by Blizzard. They have not released an official API for the game in the same way they have with some of their other games such as World of Warcraft or Starcraft II. Instead, this API is available on the Mashape Marketplace. The marketplace offers free, freemium, and paid APIs, though the Hearthstone API is free to use.

Mashape Setup

If you've never used an API hosted by Mashape before, it is a web platform for APIs. To get started using the API, you'll want to create a Mashape account (which is free) either through connecting it to your Github or through an email.

Mashape API Keys

Once you have a Mashape account, the next step is to start following the API. Instead of getting an API key for each API you are following, Mashape allows applications to utilize multiple APIs with one key. Mashage gives you a 'Default Application' that you can use to get started quickly or you can create an application specific to your project. Either way, head into the application and you'll find a 'Get The Keys' button that gives you two keys, one for production and another for testing.

While you are able to easily regenerate API keys if you see malicious activity, it is probably best to assign the key to a variable in a separate file and include that file in any pages you are working with. With that in mind, whenever you see "YOUR_API_KEY_HERE" in the example code, that means you should put your unique API key in place of that.
