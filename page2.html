Hearthstone API Endpoints

Now that we are set up with Mashape, let's take a look at how to get information back that we can use. First, we'll want to set up a new XMLHttpRequest object. Next, we'll make a GET request to the Hearthstone API. Hopefully this is familiar to you, but if it is not, we'll want to use the open method, passing it the DOMString method (i.e. GET) and the DOMString URL, which will be in the form https://omgvamp-hearthstone-v1.p.mashape.com/[stuff] where stuff are additional specifications, depending on what you want to receive (see below). Most often, we'll be returning a card or cards, but the Hearthstone API also allows you to return cardbacks as well. We'll cover both examples later on. From here, the next step is to set the request header, passing the setRequestHeader method two parameters, the DOMString header and DOMString value. In other words, we'll need to define the type of key, in this case it will always be in the form 'x-mashape-key' and the actual key itself. Once these lines have been written, we'll send the request to the server. Your code should look similar to the code below.

        var req = new XMLHttpRequest();

        req.open('get','https://omgvamp-hearthstone-v1.p.mashape.com/cards/EX1_572');

        req.setRequestHeader('x-mashape-key','YOUR_API_KEY_HERE');

        req.send();

Endpoints

Before we examine what the API returns, let's go over the different endpoints we have to work with. Here is an overview of all the endpoints offered, which you can look at in more detail in the documentation.

Endpoint Definition Description
All Cards /cards Returns all Hearthstone cards
Card Backs /cardbacks Returns all of the Hearthstone cardbacks
Card Search /cards/search/{name} Returns a list of all cards that contain the searched word in their name
Card Set /cards/sets/{set} Returns all Hearthstone cards in the given set
Cards by Class /cards/classes/{class} Returns all Hearthstone cards of the given class
Cards by Faction /cards/factions/{faction} Returns all Hearthstone cards of a given faction
Cards by Quality /cards/qualities/{quality} Returns all Hearthstone cards of the given quality (i.e. Legendary, Epic, etc.)
Cards by Race /cards/races/{race} Returns all Hearthstone cards of the given race (i.e. Pirate, Beast, etc.)
Cards by Type /cards/types/{type} Returns all Hearthstone cards of the given type (i.e. Minion, Spell, etc.)
Info /cards/info Returns the current patch, classes, sets, types, factions, qualities, races, and locales
Single Card /cards/{name} Returns the Hearthstone card by name or ID
